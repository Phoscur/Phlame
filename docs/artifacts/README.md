# Artifacts

Historical pieces rescued from the old `phlame` monorepo (GitLab: phoscur/phlame) before
its local teardown — museum items and design references, not living documents.

- [actions.md](./actions.md) — early action/entity design notes (~2019–2022): entity
  creation/update matrix (Player, Galaxy, Solarsystem, Planet, Moon, Shipyard, Fleet),
  player-control scenarios with rough sizing, and a three-step game loop sketch.
  Feeds M1 (queues) and the 2.0 vision (fleets, galaxy) — see PLAN.md.
- [ugamela-loader.json](./ugamela-loader.json) — a UGamela game-state dump (user +
  planets with metal/crystal/deuterium, buildings, techs; 2018/2019 era). Historical
  reference for the data shapes Phlame descends from; the balancing explicitly moves
  away from it (PLAN M2, docs/history.md).
